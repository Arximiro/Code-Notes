Code Notes

Flexbox

Flexbox is a display mode in CSS3 to allow for positioning of elements.

CSS Grid

CSS Grid is a display mode in CSS3 to allow for advanced 2D positioning of elements.

CSS

CSS or Cascading Style Sheets is the markup language used for styling the Internet.

HTML

HTML of Hypertext Markup Language is used to build the skeleton of a web page or app.

JavaScript

JavaScript is the most popular programming language around. It's used for the front end and the back end both.

Git

Github is used for source control and code management. It's also a great way to manage versioning

React/Redux

A JavaScript library for creating user interfaces. Redux is a state management library

VueJS

A JavaScript library that is sort of a combination between Angular and React.

BackboneJS

An older JavaScript Library that uses models and views.

SASS

AKA Sassy CSS, is a library that allows for a lot of user experience enhancements to CSS.
